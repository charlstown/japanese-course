# Verbos

## [ です ] Ser o no ser

En japonés existen verbos negativos, en este caso podemos ver el negativo del verbo ser.

| Verbo | Positivo | Negativo |
| - | - | - |
| ser/estar | です desu | でわありません　de wa arimasen |
| querer* |  | - |
| tener | います | いません | 


＊*Wa arimasen va solo con verbos y con ser o estar*
